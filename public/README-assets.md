# Static assets

Place these files in this folder so the app can show the SHAED branding:

- **SHAED Logo - Updated.png** (or **SHAED Logo.svg**) – used in the app header and on the LOI letterhead
- **SHAED Icon - Updated.png** – browser tab favicon (also used as logo fallback if the logo file is missing)

If no image loads, the app will show the word "SHAED" in teal as a fallback.
